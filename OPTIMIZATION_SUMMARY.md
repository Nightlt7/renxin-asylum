## 一、新增功能

### 界面系统
| 新增 | 说明 |
|------|------|
| 案卷档案页头 | 红十字图标 +「案卷编号 #2016-1106」+ 暗红渐变装饰线 |
| 三级按钮系统 | `btn-primary`（暗红渐变）/ `btn-secondary`（暗灰）/ `btn-ghost`（透明） |
| 统一输入框 | `input-asylum` 暗底聚焦红光 |
| 统一卡片 | `card-asylum` 渐变背景 + 柔和边框 + 深阴影 |
| 纸张纹理 | `.paper-texture` SVG 噪点纹理 + `.narrative-block` 横线格 |
| 章节标题重设计 | 红色标签式编号 + 装饰菱形分隔线 |

### 交互系统
| 新增 | 说明 |
|------|------|
| **第 2 章手机交互** | 完整手机 UI（短信气泡对话 + 通话记录 + 语音留言播放），线索融入手机操作 |
| **第 3 章地图档案柜** | 平面图热点探索 + 分类档案文件夹，双视图自由切换 |
| **线索袋折叠** | 线索按来源自动分组为可折叠的「证据袋」，展开即自动收集 |
| **线索自动收集** | 展开线索袋 / 打开手机消息 / 点击档案文件夹，自动收集无需额外按钮 |
| **章节回顾面板** | 每章谜题解答后弹出「本章关键发现」总结，手动关闭后继续 |
| **AI 语音朗读** | 已添加后删除——合成语音效果生硬，破坏沉浸感 |

### 视觉演出
| 新增 | 说明 |
|------|------|
| **开场序幕** | 红十字 → 标题 →「进入调查」，每次新会话播放一次 |
| **章节通关 CG** | 8 章各有限定动画短片（大门打开 / 报纸展开 / 手机来电 / 档案解密 / 蛇杖密码 / 病患真相 / 时钟倒转 / 天平倾斜），2.5 秒 |
| **章节过渡增强** | 三阶段渐入渐出 + 四角胶片框装饰 |
| **指认确认弹窗** | 选中嫌疑人后弹出确认面板，防止误操作 |
| **谜题红光脉冲** | 解答成功时谜题区暗红闪光 |
| **线索相机闪光** | 关键线索发现时屏幕闪白 |
| **Canvas 动态背景** | 三种场景模式：雨夜（雨滴+闪电）/ 走廊灯光（荧光灯闪烁）/ 真相揭示（暗红脉冲+金色粒子） |

| **胶片颗粒叠加** | SVG fractalNoise 全屏叠加 |
| **浮动粒子** | 18 个微型光点随机飘动 |

### 音效系统
| 新增 | 说明 |
|------|------|
| 环境音持续循环 | 雨声（粉红噪声）/ 风声（棕色噪声）/ 心跳渐强，按章节自动切换 |

| 音乐音色柔化 | 三角波/金属波 → 正弦波，音量降低 6-12dB，紧张音 30%→10% |

## 二、删除内容

| 删除 | 原因 |
|------|------|
| `canvas-confetti` 烟花效果 | 与悬疑氛围不搭，替换为低调红光脉冲 + 金色光晕 |
| AI 语音合成（`useSpeech.ts`） | 浏览器中文语音僵硬机械，严重破坏沉浸感 |
| `App.css`（185 行） | Vite 模板遗留，完全未使用 |
| `p_intro` / `p_phone` manual 类型 | 「点继续」无推理价值，改为问答题 |
| `p_jigsaw_news` 3×3 拼图 | 改为 2×3（6 块），降低操作负担 |
| IC 卡文字碎片拖拽 | 改为病房照片拼图 + 问答题，更直观 |
| 线索「收集」按钮 | 展开线索袋即自动收集 |
